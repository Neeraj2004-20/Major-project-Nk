---

### Step 2: Deploy on Render.com (2 minutes)

1. **Sign Up**: Go to https://render.com → Sign up with GitHub

2. **Create Web Service**:
   - Click **"New +"** → **"Web Service"**
   - Select repository: **stocksense-ai**
   - Name: `stocksense-ai` (already configured in render.yaml)
   - Click **"Create Web Service"**

3. **Wait for Build**: ~10-15 minutes ☕

4. **Get Your URL**:
   ```
   https://stocksense-ai.onrender.com
   ```
