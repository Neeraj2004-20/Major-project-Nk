# 📊 Prediction Tracker & PDF Reports Guide

## Overview
Two powerful production-ready features added to track AI prediction accuracy and generate professional PDF reports.

---

## 🎯 Feature 1: Prediction Accuracy Tracker

### What It Does
- **Automatically logs** every prediction made by the AI model
- **Verifies predictions** against actual market prices using real-time data from Yahoo Finance
- **Calculates accuracy metrics** including overall accuracy, average error, and direction accuracy
- **Categorizes predictions** into:
  - 🎯 **Excellent**: Error < 2%
  - ✅ **Good**: Error < 5%
  - ↗️ **Direction Correct**: Predicted direction was right
  - ❌ **Incorrect**: Direction prediction was wrong
  - ⏳ **Pending**: Waiting for actual price data

### How It Works

#### Backend (`prediction_tracker.py`)
```python
class PredictionTracker:
    def log_prediction(symbol, current_price, predicted_price, ...)
    def verify_predictions()  # Fetches actual prices
    def get_accuracy_stats()  # Returns metrics
    def get_all_predictions()  # Returns history
```

#### API Endpoints
- `GET /predictions/accuracy` - Get overall accuracy statistics
- `GET /predictions/history?limit=20` - Get recent prediction history

#### Frontend Features
- **📊 Accuracy Dashboard** tab with:
  - Real-time accuracy metrics (Overall Accuracy, Avg Error, Direction Accuracy)
  - Historical predictions table with status indicators
  - Color-coded status badges
  - One-click refresh functionality

### Data Storage
All predictions stored in `predictions_log.json`:
```json
{
  "symbol": "RELIANCE.NS",
  "prediction_date": "2026-01-12T10:30:00",
  "current_price": 2850.50,
  "predicted_price": 2920.75,
  "actual_price": 2905.30,
  "days_ahead": 7,
  "confidence": 0.75,
  "status": "good",
  "error_pct": 0.53
}
```

## 📄 Feature 2: PDF Report Generator

### What It Does
- Generates **professional PDF reports** with all AI insights
- Two report types:
  - **Prediction Report**: Detailed analysis of a specific prediction
  - **Accuracy Report**: Overall performance summary with all predictions

### Report Contents

#### Prediction Report Includes:
1. **Header**: "AI Market Prediction Report" with company logo
2. **Prediction Summary Table**:
   - Symbol, Prediction Date, Current Price, Predicted Price
   - Days Ahead, Price Change %, Confidence Score
3. **AI Explanation Section**:
   - Natural language summary
   - Key factors (Technical Indicators, Market Sentiment, Volume Analysis)
4. **Market Health Analysis**:
   - Overall score
   - Detected anomalies
5. **Risk Scenarios**:
   - Best Case, Worst Case, Most Likely, Black Swan, Market Correction
   - Each with probability, target price, expected change
6. **Professional Disclaimer**
7. **Footer**: "Generated by AI Market Predictor | Powered by Advanced Transformer Neural Networks"

#### Accuracy Report Includes:
1. **Header**: "Prediction Accuracy Report"
2. **Performance Metrics Table**:
   - Total Predictions, Overall Accuracy Rate, Avg Error %, Direction Accuracy
3. **Status Breakdown**:
   - Excellent/Good/Direction Correct/Incorrect counts
4. **Recent Predictions Table** (All historical predictions)
5. **Analysis Summary** with insights
6. **Professional Disclaimer**

### How It Works

#### Backend (`report_generator.py`)
```python
class PDFReportGenerator:
    def generate_prediction_report(report_data)
    def generate_accuracy_report(stats)
```

#### API Endpoints
- `POST /report/generate` - Generate PDF report
  - Body: `{ "report_type": "prediction" | "accuracy", "symbol": "RELIANCE.NS" }`
  - Returns: `{ "filename": "...", "download_url": "/report/download/..." }`
- `GET /report/download/{filename}` - Download generated PDF

#### Frontend Features
- **Two download buttons** in Accuracy Dashboard:
  - 📄 Download Prediction Report (PDF)
  - 📊 Download Accuracy Report (PDF)
- Automatic file download on click
- Success/error notifications

### Report Storage
All reports saved in `reports/` folder:
- `prediction_report_SYMBOL_YYYYMMDD_HHMMSS.pdf`
- `accuracy_report_YYYYMMDD_HHMMSS.pdf`

## 🚀 Usage Guide

### Step 1: Make a Prediction
1. Open http://localhost:8000
2. Select a company (e.g., Reliance Industries)
3. Choose prediction period
4. Click "🔮 Get AI Prediction"
5. **Prediction automatically logged** to tracker

### Step 2: View Accuracy Dashboard
1. Click **📊 Accuracy Dashboard** tab
2. Click **🔄 Refresh Accuracy Data** button
3. View metrics:
   - Overall Accuracy Rate
   - Average Error Percentage
   - Direction Accuracy
   - Total Predictions
4. Scroll down to see **Recent Predictions** table

### Step 3: Generate PDF Reports
1. In Accuracy Dashboard, scroll to "📄 Generate Reports" section
2. Click either:
   - **📄 Download Prediction Report** (for latest prediction with full AI analysis)
   - **📊 Download Accuracy Report** (for overall performance summary)
3. PDF will automatically download

### Step 4: Verify Predictions (Automated)
- The system automatically checks actual prices when you refresh accuracy data
- Uses Yahoo Finance to get real market data
- Compares predicted vs actual prices
- Calculates error percentage
- Updates prediction status

## 📊 Accuracy Metrics Explained

### Overall Accuracy Rate
- **Calculation**: (Excellent + Good + Direction Correct) / Total Predictions
- **Meaning**: Percentage of predictions that were at least directionally correct
- **Good Performance**: > 60%

### Average Error Percentage
- **Calculation**: Mean of `|predicted - actual| / actual * 100`
- **Meaning**: How far off predictions are on average
- **Good Performance**: < 5%

### Direction Accuracy
- **Calculation**: Correct up/down predictions / Total Predictions
- **Meaning**: How often the AI correctly predicts if price goes up or down
- **Good Performance**: > 70%
